Relógio Eletrônico de Ponto

Projeto desenvolvido para definir as rotas dos vigias, utilizando relógios eletrônicos baseados em RFID. Cada vigia possui um cartão RFID que deve ser registrado periodicamente. As leituras definem a rota de monitoramento, que pode ser consultada pelo gestor com base na data, horário, local e ID do cartão.

Tecnologias: RFID, ESP32, MySQL
